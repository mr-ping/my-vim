# my-vim

## Features:
- auto completion with virtual environment
- syntastic checking
- func bar on the bottom
- file tree

## Plugin Manager:
plug-vim

## Pluggers
- YouCompleteMe
- syntastic
- vim-airline
- nerdtree

## How to use

```bash
cd ~
rm -rf .vim .vimrc
git clone git@github.com:mr-ping/my-vim.git .vim
```

Execute `PlugClean` and `PlugInstall` commands in VIM.
